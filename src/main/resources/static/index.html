Features

🔍 Advanced Job Search

Search jobs with multiple filters including location, experience, salary range, and job type.

📝 Resume Management

Create textual resumes and upload formatted documents (PDF/DOCX) up to 2MB.

🚀 One-Click Apply

Apply to jobs instantly with your saved resume and optional cover letter.

📊 Application Tracking

Track your application status in real-time with detailed updates.

🏢 Employer Dashboard

Manage job postings, view applicants, and track hiring statistics.

🔔 Notifications

Receive in-app notifications for application updates and job recommendations.

API Documentation

Base URL

http://localhost:8080/api

Authentication

Include JWT token in Authorization header:

Authorization: Bearer <your-jwt-token>

Authentication Endpoints

POST /auth/register/jobseeker

Register as a job seeker

POST /auth/register/employer

Register as an employer

POST /auth/login

Login to get JWT token

Job Endpoints

GET /jobs/search?title=...&location=...

Search jobs with filters

GET /jobs/active

Get all active jobs

POST /jobs

Create a new job (Employer only)

Application Endpoints

POST /applications

Apply for a job

GET /applications/my-applications

Get your applications

PATCH /applications/{id}/status

Update application status (Employer only)

About RevHire

RevHire is a comprehensive job portal application built with Spring Boot, MySQL, and JWT authentication. It provides a complete solution for job seekers to find opportunities and employers to discover talent.

Technology Stack:

Spring Boot 4.0.2 | Java 17 | MySQL 8.0 | JWT | Log4j2 | JUnit 5
